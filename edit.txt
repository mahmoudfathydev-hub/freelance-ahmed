تعديل النص في Hero Section (شرح بسيط جدًا)

الفكرة العامة:
الموقع معمول بـ React / Next.js
كل جزء في الصفحة له ملف خاص به
إنت هتعدّل النص فقط، مش الستايل

الخطوات:

1- حدّد الصفحة
إنت شغال على الصفحة الرئيسية (Home)

2- افتح ملف الهيرو
المسار:
app/home/Hero.tsx

3- افتح الملف ودوّر على كلمة:
return

أي حاجة تحت return دي HTML عادي

4- عدّل النص فقط
مثال:
<h6>Available for new projects</h6>
<h2>WEBSITE <br /> DEVELOPER</h2>
<p>Welcome to my digital world!</p>

إنت بتعدّل الكلام اللي بين التاجات بس

5- ممنوع تلمس:
- className
- data-aos
- أي حاجة ليها علاقة بالستايل

مثال ممنوع:
className="w-full bg-[#181A1B]"

مثال تعديل صح:
قبل:
<h2>WEBSITE <br /> DEVELOPER</h2>

بعد:
<h2>FRONTEND <br /> DEVELOPER</h2>

الخلاصة:
- افتح الملف
- دوّر على return
- عدّل الكلام اللي جوه التاجات
- متلمسش الستايل
- لو مش فاهم حاجة، سيبها زي ما هي
